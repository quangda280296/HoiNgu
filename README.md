# Hỏi ngu

## Description
Game vui nhộn với gói câu hỏi thú vị và đáp án bất ngờ, có thưởng vòng quay nếu người chơi trả lời đúng nhiều câu, mục đích chính là giúp người chơi thư giãn, giải trí.
